# 基于 i.MX6NULL 的 Linux 驱动开发与多媒体应用平台

> 本项目是以 I.MX6ULL 为硬件平台，基于 Qt 搭建的用于 Linux 各类设备驱动及应用的综合测试平台，包括实现 GPIO、I2C、SPI、ADC 等驱动统一调试界面，并通过虚拟设备模拟驱动接口实现跨平台运行

## 硬件
- UVC 相机 
- 画图板
- 姿态角展示
- 环境光监测 
- ADC 采集
- 设置 
- CAN 接收测试
- UART 接收测试
- 音乐播放器
- 视频播放器

## 实现功能
- CAN通信测试
  - 接收从机消息
  - 单机Send按钮发消息给从机
- 画图板
  - 支持任意颜色绘制
  - 鼠标状态下支持滚轮调节线条粗细
  - 单击保存按键可以将绘制的图像保存到本地
  - 橡皮擦除线条
  - 单击扫把按键可以一键清除画布绘制内容
  - 单击相册可以查看保存的画布图像，支持长按放大显示
- 支持UVC相机
  - 单击打开按键打开相机
  - 单击拍照即可拍摄照片
  - 单击相册查看保存相片
  - 支持长按放大显示
- 接触测试和环境光监测
  - 利用 AP3216C 
- ADC
  - 可采集 GP101 1001引脚输入的电压
  - 单击图例可显示/隐藏曲线
- 支持UART通信测试
  - (目前仅有CH34x驱动)
- 音乐播放器
  - 可从文件管理器选择要播放的音乐
  - 支持拖动进度条改变歌曲进度
  - 可以调节喇叭音量大小，修改完毕后两秒后自动隐藏
  - 支持循环、乱序、单曲循环三种播放模式
  - 支持黑胶唱片播放、暂停动态效果
- GPIO控制外设
  - 单击LED 图标打开/关闭开发板上的LED 灯
  - 单击 Beep图标打开/关闭开发板上的蜂鸣器
- 支持调节屏幕亮度
- 视频播放
  - MP4格式文件

## 项目介绍

### 源码结构

- IMx6ULL.pro
  - 项目管理文件，这个项目管理文件十分重要，当您加入了文件或者删除了文件， Qt Creator 会自动修改这个*.pro 文件。*
  - 有时候需要打开这个.pro 文件添加我们的设置项    
- Headers目录
  - 存放的是项目内所有的头文件*.h    
- Sources目录
  - 存放的是项目内的所有 C++源码文件*.cpp    
- Forms目录
  - 放项目内所有界面文件*.ui。
  - *.ui 文件由 XML 语言描述组成，编译时会生成相应的 cpp 文件，这样交叉编译器就可以编译它了    
- Resources目录



### 核心控制与 UI 框架 

| **文件名**           | **作用描述**                                                 |
| -------------------- | ------------------------------------------------------------ |
| **main.cpp**         | **程序入口**。负责实例化 `QApplication`，加载全局样式，并启动主界面。 |
| **mainwindow.cpp**   | **主导航界面**。通常是一个桌面或菜单，负责切换不同的功能模块（如点击图标打开音乐播放器）。 |
| **config.cpp**       | **配置管理**。存放路径设置、字体大小、屏幕分辨率等全局静态变量。 |
| **LayoutSquare.cpp** | **UI 布局助手**。可能用于实现主界面的网格（Square）排版逻辑。 |



### 硬件接口层 (与底层的桥梁) 

这些文件通常通过 QFile 或标准 C 接口读写 Linux 设备文件（如 /dev/xxx 或 /sys/class/xxx）。
| 文件名       | 对应的硬件/协议                               |
| :----------- | :-------------------------------------------- |
| adc.cpp      | 电位器/电压检测 (Analog-to-Digital Converter) |
| gpio.cpp     | 控制 LED 灯或读取按键 (General Purpose I/O)   |
| uart.cpp     | 串口通信 (RS232/RS485)                        |
| can.cpp      | CAN 总线通信 (工业常用)                       |
| ap3216c.cpp  | 环境光 & 接近传感器 (I2C 接口)                |
| icm20608.cpp | 六轴加速度计 & 陀螺仪 (SPI 接口)              |
| ble.cpp      | 蓝牙低功耗 (Bluetooth Low Energy)             |
